# Immagini condivise HeroZone

Queste immagini sono usate da TUTTI i giochi HeroZone.
Catturale UNA sola volta con il coord-finder.

## Immagini da catturare

| File                  | Cosa catturare                              |
|-----------------------|---------------------------------------------|
| arrow_back.png        | Il bottone < in alto a sinistra del launcher|
| game_grid.png         | Un pezzo della griglia (es. 2-3 tile vuoti) |
| difficulty_normal.png | Il tab "NORMAL" nella schermata del gioco   |
| btn_launch_game.png   | Il bottone blu "LAUNCH GAME"                |
| btn_quit.png          | Il bottone "QUIT" (durante gioco attivo)    |

## Come catturarle

1. Apri HeroZone
2. Lancia il coord-finder in modalità immagine:
   
   node scripts/capture-herozone-shared.js

3. Segui le istruzioni per catturare ogni immagine
